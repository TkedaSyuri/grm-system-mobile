# Guest Room Monitor（リネン管理システム）アプリ版

ビジネスホテル向けの **客室状況管理・清掃業務支援・連絡システム** です。
フロントスタッフと清掃員の間でリアルタイムに情報共有を行い、ホテル業務の効率化を実現します。

## 概要
本アプリは、主に清掃員が清掃中に利用することを想定したモバイルアプリです。
客室の清掃状況や作業内容の更新、フロントとのリアルタイムな連絡をシームレスに行えます。

## 主な機能

| 機能 | 説明 |
| :--- | :--- |
| **客室状況の確認** | 客室ごとのステータス（清掃中・完了など）をリアルタイムで確認可能 |
| **客室状況の変更** | 現場の清掃状況に合わせて、即座に部屋のステータスを更新 |
| **相互連絡機能** | フロントスタッフと清掃員の間でメッセージをリアルタイムに送受信 |
| **認証機能** | 権限に基づいたユーザー認証により、セキュアな業務管理を実現 |

## 操作方法

### 1. 客室状況の変更
現場の進捗に合わせて、以下の手順で部屋のステータスを変更します。
* **操作:** 対象の部屋番号を**長押し**します。
* **更新:** 出現した「ステータス変更ボタン」をタップし、現在の状況を選択して変更を完了します。

### 2. タスク管理
清掃に伴うタスク（備品補充など）の完了報告を行います。
* **タスク完了:** 該当タスクを**左にスライド**し、右側に出現する「完了ボタン」をタップします。
* **完了済みの確認:** 右上の「完了したタスク」メニューから履歴を確認できます。
* **未完了に戻す:** 完了済みリスト内のタスクを**右にスライド**し、左側に出現する「元に戻す」をタップします。

### 3. リアルタイム連絡
フロントスタッフとの相互連絡にはチャット機能を利用します。
* 掲示板形式またはダイレクトメッセージにより、清掃中のトラブル報告や指示受けをリアルタイムに行うことが可能です。
